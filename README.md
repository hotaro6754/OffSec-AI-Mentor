## üöÄ Quick Start

### Prerequisites
- Node.js v18+
- Groq API Key ([console.groq.com](https://console.groq.com))

### Installation
1. `git clone https://github.com/hotaro6754/OffSec-AI-Mentor.git`
2. `npm install`
3. Create a `.env` file and add `GROQ_API_KEY=your_key`
4. Run: `npm start`
5. Open: `http://localhost:3000`
